Приложение для управления списком сотрудников на Python
В современном мире управление списком сотрудников является важной задачей для любой организации. Это позволяет отслеживать данные о сотрудниках, такие как их имена, должности, контакты и заработная плата.

Чтобы облегчить эту задачу, можно использовать специальное приложение. В этой статье мы рассмотрим простое приложение для управления списком сотрудников, написанное на языке Python.

Описание приложения
Приложение представляет собой простую программу с графическим интерфейсом пользователя (GUI), написанную с использованием библиотеки Tkinter. Оно хранит данные о сотрудниках в базе данных SQLite.

Приложение обеспечивает следующий набор функций:
Добавление нового сотрудника
Редактирование существующей записи сотрудника
Удаление записей о сотрудниках
Поиск сотрудников по полному имени
Отображение данных в виджете Treeview
Запуск приложения
Чтобы запустить приложение, убедитесь, что у вас установлен Python. Затем следуйте этим шагам:
1. Склонируйте репозиторий или загрузите файлы проекта.
2. Если необходимо, установите библиотеки, используя pip:
shell
pip install tkinter
3. Запустите основной файл main.py:
shell
python main.py
Классы и их функциональность
Приложение использует следующие классы:
Main - главный класс, который представляет главное окно приложения и отвечает за отображение списка сотрудников, добавление, редактирование, удаление и поиск записей. Он также инициализирует и управляет базой данных SQLite.
Child - класс, который открывает окно для добавления нового сотрудника и позволяет ввести данные о сотруднике, включая полное имя, номер телефона, адрес электронной почты и заработную плату. Введенные данные сохраняются в базе данных.
Update - класс, который открывает окно для редактирования существующей записи сотрудника и позволяет изменять полное имя, номер телефона, адрес электронной почты и заработную плату сотрудника. Он также обновляет запись в базе данных.
Search - класс, который открывает окно для поиска сотрудников по полному имени и позволяет пользователям ввести полное имя для поиска. Результаты поиска отображаются на основе введенного имени.
DB - класс, который управляет базой данных SQLite и создает таблицу для хранения данных о сотрудниках.
